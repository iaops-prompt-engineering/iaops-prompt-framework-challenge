# Justificativa

O Context apresenta o padrao interno de IaC: tags obrigatorias, prefixo `hvt-`, criptografia, versioning, block public access e logging.
O Action pede a criacao de um modulo Terraform reutilizavel com `main.tf`, `variables.tf`, `outputs.tf` e exemplo.
O Result define codigo limpo, padronizado e consumivel por outros times; o Example usa o modulo VPC como referencia de estilo.
