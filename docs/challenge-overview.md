# Visao Geral do Desafio

Este repositorio registra a resolucao das 8 questoes do desafio de Prompt Engineering usando a empresa ficticia Hill Valley Tech como contexto.

## Sistemas

- Chronos: API gateway e plataforma core.
- Ledger: data warehouse PostgreSQL com historico de transacoes e eventos.
- Reactor: processamento assincrono por filas.
- Beacon: observabilidade, metricas, logs e alertas.
- Lift: produto beta em amadurecimento.

## Personagens

- Doc Brown: CTO.
- Jennifer Parker: PM.
- Lorraine Baines: lider SRE.
- George McFly: engenheiro senior do legado.
- Goldie Wilson: CEO.
- Strickland: seguranca e compliance.

## Objetivo

Para cada cenario, foi criado um prompt aplicando o framework indicado, seguido do output gerado e da justificativa que mostra como os componentes do framework aparecem no prompt.
